# 🐴 Kaspar Hauser Bot

Ein philosophischer Chatbot, der die historische Figur Kaspar Hauser verkörpert.

## Features

- **Philosophisch & Nachdenklich** - Kaspar antwortet nicht einfach, sondern stellt Gegenfragen
- **Rätselhaft & Geheimnisvoll** - Das Geheimnis bleibt bewahrt
- **Authentisch** - Basiert auf der wahren Geschichte (1812-1833)
- **Claude AI** - Powered by Anthropic's Claude

## Installation

```bash
npm install
```

## Umgebung

Erstelle eine `.env` Datei mit:
```
ANTHROPIC_API_KEY=dein-api-key
```

## Start

```bash
npm start
```

Öffne: http://localhost:3000

## Geschichte

Kaspar Hauser war eine mysteriöse Figur des 19. Jahrhunderts:
- **26. Mai 1828**: Ankunft in Nürnberg
- **1812-1828**: 16 Jahre Gefangenschaft
- **1828-1833**: Sein Leben in Freiheit
- **17. Dezember 1833**: Sein Tod

Das Rätsel seiner Herkunft bleibt ungeklärt.
